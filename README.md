💡 Controle de Frequência do LED com Botões (Arduino)
📌 Descrição

Este projeto demonstra como controlar a frequência de piscar de um LED utilizando botões (teclas) com Arduino.

O usuário pode alterar a velocidade do LED em tempo real, tornando o sistema interativo e ideal para aprendizado de controle digital e lógica de programação.

A frequência representa quantas vezes o LED liga e desliga por segundo, sendo um conceito fundamental em eletrônica e sistemas embarcados

🚀 Funcionalidades
Controle da velocidade do LED com botões
Aumento e diminuição da frequência de piscada
Interação em tempo real
Uso de lógica condicional (if/else)
Introdução ao conceito de frequência
🧰 Componentes utilizados
Arduino Uno
LED
Resistores (220Ω)
Botões (push buttons)
Protoboard
Jumpers
🔌 Esquema de ligação (exemplo)
LED
Anodo → pino digital (ex: 13)
Catodo → resistor → GND
Botões
Um lado → pino digital
Outro lado → GND (ou VCC dependendo da lógica)
